最近一段时间都在学习Java的相关知识，给我的感觉是它涉及的范围广内容多以及功能强大，所以学习起来周期蛮长的而且还挺费劲。包括语法基础，面向对象，常用工具类，文件操作，网络与多线程，数据操作，web相关，最后就是不同的框架，比如Hibernate,Mybatis,Struts2,Spring,Spring MVC等等。内容很多，所以除了语法和数据库，以及jsp，servlet，JDBC，其他的也只是先了解下，后面项目做熟悉起来，就慢慢研究。

今天做了一个APP的数据接口小项目，主要涉及的部分有以下几个方面：

2.添加mysql-connector-java-5.1.46.jar GSON-2.3.1.jar Servlet.jar 配置web.xml文件

3.JDBC读取数据库，封装Model，然后输出即可

1.JsonServlet.java

2.News.java
