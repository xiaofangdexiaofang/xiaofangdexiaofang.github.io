因为面试的时候，大家都对这个比较感兴趣，由于本人记性不好，回答的不全面，所以还是写下来，后面也方便。接触逆向是在18年10月份，到19年3月份为止就不做iOS逆向了，主要是所在地区招iOS逆向的很少，基本没有岗位招人做这方面。主要是以下的几个流程，感觉每个步骤都有很多的坑😂，很多东西都想不起来，这边会慢慢加

从APPStore下载的需要砸壳，这边直接省略掉这个步骤，从PP助手里面下载是直接脱壳的

脱壳

  • otool -l 可执行文件 | grep crypt 这个命令可以查看ipa包是否加壳
  • otool -L DingTalk 依赖的库查询：系统库、支持的架构信息及版本号
  • otool -tV DingTalk 汇编码 使用 otool -tV ex 则整个ARM的汇编码就都显示出来了
  • otool -l DingTalk | grep -B 2 crypt 或者 otool -l xxx | grep crypt查看该应用是否砸壳
  • otool -h DingTalkMach-O头结构等
  • 参考iOS逆向-手把手教你写支付宝蚂蚁森林手机能量助手
  • 参考狗神和庆哥的书

去掉__RESTRICT section步骤 和 ptrace两个保护

  • ptrace 主要是防止加载调试器，__RESTRICT section 会妨碍cycript的使用
  • 参考iOS逆向-手把手教你写支付宝蚂蚁森林手机能量助手
  • 参考狗神和庆哥的书

解决去掉 __RESTRICT section 和 ptrace 闪退问题

  • 替换掉__RESTRICT section后，保存即可，然后替换掉里面的二进制文件
  • 最主要一步就是删除。app中的codesing签名的文件夹
  • 请参考这个帖子

获取头文件，使用这个工具class-dump

  • class-dump -H AlipayWallet.decrypted -o ./

使用cycript命令去调试打印

  • Cycript是Objective-C++,ES6,Java等语法的混合物，用来修改调试正运行在内存中的app的
  • 具体参考文档
  • 或者可以使用MJ封装的mjcript
  • 可以实时的打印出当前调试的值

使用THEOS自带的logify.pl生成对应类的全部方法调用和参数

  • /opt/theos/bin/logify.pl xxxx.h > Tweak.xm
  • 编译Tweak.xm 注意：有编译报错，有些类和代理block是不认识的 出错的删除掉即可
