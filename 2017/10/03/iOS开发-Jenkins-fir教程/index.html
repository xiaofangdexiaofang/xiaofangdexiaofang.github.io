iOS开发 Jenkins+fir教程

发表于 2017-10-03 | 分类于 自动化
字数统计: 583 | 阅读时长 ≈ 2

记录安装问题和解决办法，防止再次踩坑

由于Jenkins是基于java开发，首先要安装SDK（https://java.com/zh_CN/）

安装Jenkins

直接下载安装（http://jenkins-ci.org/）

使用命令行安装，前提是你已经安装homebrew

brew install jenkins

安装完成，你会看到这句话： please use the following password to proceed to installation:记住下面的密码

打开Jenkins网页（[http://localhost:8080/]

你会看到下面这个界面，输入刚才保存的密码。若是直接下载安装的，请自行百度搜索如何找到密码。

看到下面直接输入用户名和密码以及邮件

安装GitLab插件

因为我们用的是GitLab来管理源代码，Jenkins本身并没有自带GitLab插件，所以我们需要依次选择 系统管理->管理插件，在“可选插件”中选中“GitLab Plugin”和“Gitlab Hook Plugin”这两项，然后安装。

安装Xcode插件

同安装GitLab插件的步骤一样，我们依次选择系统管理->管理插件，在“可选插件”中选中“Xcode integration”安装。

安装fir.im插件 系统设置->管理插件->高级 上传fir.im的插件

###创建一个新的任务 ->构建一个自由风格的软件项目

General

源码管理

点击Add，添加对应的用户名和密码

添加构建时间
